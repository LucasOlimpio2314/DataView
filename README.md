# 📊 DataView

## 📌 Sobre o Projeto

**DataView** é uma aplicação em desenvolvimento que visa facilitar a análise de dados a partir de arquivos Excel (.xlsx, .csv). A aplicação permitirá o upload desses arquivos, processamento dos dados e a exibição de dashboards interativos.

Atualmente, a equipe está trabalhando na configuração inicial do projeto, definindo a arquitetura e implementando os primeiros componentes do backend e frontend.

## 🚀 Objetivo do Projeto

Criar uma ferramenta simples e eficiente para análise de dados, permitindo que usuários enviem arquivos Excel e visualizem gráficos gerados automaticamente.

## 🎯 Funcionalidades Planejadas

✔️ **Upload de Arquivos Excel (.xlsx, .csv)**  
✔️ **Processamento de Dados com Pandas**  
✔️ **Geração de Dashboards Interativos (Matplotlib, Seaborn, Plotly)**  
✔️ **API para Comunicação entre Backend e Frontend**  
✔️ **Interface Web para Upload e Visualização**  
✔️ **Validação de Arquivos e Tratamento de Erros**  
✔️ **Autenticação de Usuário (Opcional - Fase Avançada)**  

---

## 🛠 Tecnologias Planejadas

**Backend:** Python, Flask, Pandas  
**Frontend:** HTML, CSS (Bootstrap/Tailwind), JavaScript  
**Visualização de Dados:** Matplotlib, Seaborn, Plotly  
**Banco de Dados:** SQLite ou PostgreSQL (se necessário)  
**Testes:** Pytest
